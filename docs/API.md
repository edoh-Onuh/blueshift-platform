# API Reference

This document provides a high-level API reference for the Blueshift programs. For full details, see each program README.

## Programs

### Flash Loan
- `borrow(borrow_amount: u64)`
- `repay()`

### Vault
- `deposit(amount: u64)`
- `withdraw()`

### Escrow
- `make(seed: u64, receive: u64, amount: u64)`
- `take()`
- `refund()`

## Accounts
Refer to each program README for required accounts and constraints.
