# Integration Guide

This guide explains how to integrate Blueshift Flash Loan, Vault, and Escrow programs into your Solana protocol.

## Quick Links
- [Flash Loan README](../blueshift_anchor_flash_loan/README.md)
- [Vault README](../blueshift_anchor_vault/README.md)
- [Escrow README](../blueshift_anchor_escrow/README.md)

## Prerequisites
- Solana CLI (v1.18+)
- Anchor (v0.29+)
- Node.js 18+

## Steps
1. Install dependencies and build the program you want to integrate.
2. Use the IDL in target/idl to generate TypeScript types.
3. Use program accounts and methods as shown in each program README.
4. Test on devnet before mainnet.
